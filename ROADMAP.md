# Roadmap

Из конкурентного разбора (Rose, Combot, CAS, Shieldy, KolasAI, SmartMod) — см. обсуждение в чате.
Порядок = приоритет.

## 1. Капча для новых участников — `готово`
Отраслевой стандарт (есть у Rose, Shieldy, SmartMod), у нас не было вообще.
**Выключена по умолчанию**, в Mini App убрана в свёрнутый раздел «Дополнительно»
(не на первом экране настроек, чтобы не перегружать базовый флоу).

- `captchaEnabled: boolean` (default `false`) в настройках группы
- При включении: новый участник мутится + получает кнопку-подтверждение
  «Я не бот» в группе (2 минуты). Не нажал вовремя — кик при следующем
  сообщении в группе (лениво, без отдельного воркера/крона — serverless-деплой
  без персистентного процесса, см. `lib/telegram/captcha.ts`)
- Чат-команда `/captcha on|off`
- Mini App: тумблер в свёрнутой секции «Дополнительно» (по умолчанию закрыта)
- Осведомлённость о правах бота (`missingPermissionsFor`) теперь учитывает
  и капчу — она тоже требует право «Блокировка участников»

## 2. Приветственные сообщения — `готово`
- `welcomeEnabled: boolean`, `welcomeMessage: string | null`
- Плейсхолдер `{user}` → HTML-упоминание нового участника
- Команда `/welcome <текст>` / `/welcome off`
- Поле ввода в Mini App (видимо на основном экране настроек, не спрятано)

## 3. Расширенная статистика (не только нарушения) — `готово`
- Новые счётчики за день: `messages`, `joins` (`lib/db/stats.ts` → `incrementActivity`/`getActivity`)
- Отдельная секция «Активность» на экране статистики Mini App, над блоком нарушений

## 4. Явное позиционирование uz-cyrl + ru — `сделано частично`
В ходе разбора рынка не нашёл ни одного конкурента, целящегося именно в
связку узбекский-кириллица + русский. Открытая ниша.

- Описание бота (`setMyDescription`) можно усилить акцентом на это
- README уже описывает локализацию как основную фичу, а не довесок

## 5. Платный тариф (freemium + PRO) — `готово`
Реализовано целиком: freemium-порог по модели Combot, выделенная Groq-квота,
капча/антирейд как PRO-фичи, оплата через Telegram Stars.

- `plan: "free" | "pro"`, `planExpiresAt` в настройках группы (`lib/billing/plan.ts`)
- **Freemium-порог**: `FREE_TIER_MAX_MEMBERS = 200` (тот же ориентир, что у
  Combot) — группы до 200 участников получают капчу/антирейд бесплатно как
  грейс-период; больше — нужен PRO. Базовая модерация (мат/спам) бесплатна
  всегда, независимо от размера — это принципиально не паяуоллим
- **Выделенная Groq-квота**: `lib/moderation/groq.ts` — RPM/RPD/TPM/TPD бюджеты
  разделены на пул `free` и пул `pro`, чтобы всплеск бесплатных групп не мог
  исчерпать квоту у платящих (сейчас условный сплит 17/8 RPM, легко подкрутить)
- **Капча + антирейд как PRO-фичи**: показывается PRO-бейдж и подсказка в
  Mini App, если группа не подходит по размеру/подписке; включение через
  API/команду отбивается 402, если не положено
- **Антирейд**: `checkRaid()` в `lib/moderation/flood.ts` — N вступлений за
  короткое окно принудительно включают капчу для всех вступающих в этот
  момент, даже если обычная капча выключена
- **Оплата — Telegram Stars** (`lib/telegram/payments.ts`), нативная
  30-дневная подписка через `createInvoiceLink` с `subscription_period`
  (это единственный метод API, который поддерживает recurring — `sendInvoice`
  не умеет). Команда `/upgrade` в чате, кнопка «Оформить PRO» в Mini App через
  `Telegram.WebApp.openInvoice()`. Цена — 349 Stars (~$5/мес, тот же ориентир,
  что и у Combot)
- Обработка `pre_checkout_query` (всегда ok, отвечаем в течение 10 сек) и
  `successful_payment` — активирует `plan: "pro"` по `subscription_expiration_date`
  из платежа, шлёт админам подтверждение
- **Не протестировано вживую** — реальный платёж через Stars можно проверить
  только внутри настоящего Telegram-клиента, локально это не симулируется

### Отраслевые пресеты словаря — тоже сделано попутно
`lib/moderation/presets.ts` — стартовые наборы скам-фраз (не мат) под нишу:
agro, ecommerce, edtech, finance. Команда `/preset <ключ>` и кнопки в Mini App
добавляют слова поверх уже имеющихся у группы, не заменяя их. Это черновые
наборы для ревью админом, не исчерпывающий список — как и с базовым
словарём мата, качество важнее охвата.

## Аудит от 2026-07-22 — исправлено

Ультра-критичный ревью всего проекта против этого roadmap нашёл и закрыл
8 реальных багов (гонки состояний в Redis, XSS-подобный сбой приветствий,
O(все группы бота) вместо O(групп юзера) в дашборде, отсутствие проверки
прав бота при сохранении канала-журнала из Mini App, 5 команд — включая
`/upgrade` — не зарегистрированных в меню Telegram, тихий пропуск
модерации при сетевом сбое) + 2 мелких (мёртвый перевод, ещё одна гонка
в Redis). Капча/антирейд/`/upgrade` были функционально готовы и раньше, но
невидимы в автокомплите "/" — теперь видимы.
